# Local E-commerce Platform

A simple e-commerce store application featuring product listings and a shopping cart.

## Features

- **Product Listing**: View available products.
- **Shopping Cart**: Add items to a cart and view the summary.
- **Backend API**: Node.js/Express backend to serve products and handle cart operations.

## Project Structure

```
task-03-ecommerce-platform/
├── client/            # Frontend files (HTML, CSS, JS)
├── server/            # Backend server
└── package.json
```

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Backend**: Node.js, Express.js

## Getting Started

### Installation

1. Navigate to the folder:
   ```bash
   cd task-03-ecommerce-platform
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Running the App

```bash
# Start the server
npm start
```

Server usually runs on `http://localhost:3000`. Open your browser to view the store.
